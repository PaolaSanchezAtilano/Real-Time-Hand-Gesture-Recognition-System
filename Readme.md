# Real-Time Hand Gesture Recognition System

A real-time hand gesture recognition application developed in Python using computer vision techniques.  
The system detects hand landmarks through the webcam and classifies specific gestures dynamically.

## Features

- Real-time hand detection using MediaPipe
- Identification of 21 hand landmarks
- Gesture classification:
  - Open hand
  - Closed fist
  - Peace sign
  - Thumbs up
- Live visual feedback on screen
